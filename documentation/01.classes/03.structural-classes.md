Structural classes provide a **semantic context** for the underlying elements and are preferably utilized to define a layout.


Syntax
------

```css
.{context}
```

```css
.{context}-{name}
```


Overview
--------

| Tag     | Example     |
|---------|-------------|
| article | .content    |
| body    | .body       |
| footer  | .footer     |
| header  | .header     |
| main    | .main       |
| nav     | .navigation |
| section | .section    |
| aside   | .sidebar    |
